==========
 Glossary
==========

.. glossary::
   :sorted:

   Backup Path
      The backup path is where your items are stored.

   GAL
      The Global Address List is a directory of e-mail addresses configured
      on a domain and available to all users.

..
      Tag
         A tag is a label assigned to e-mails with the purpose is to
         classify them in a more fine-grained way than folders do, since
         multiple tags can be assigned to an e-mail
