* update the list of packages

  .. code:: console

     # apt update

* upgrade the system

  .. code:: console

     # apt upgrade

* or execute both task at once:

  .. code:: console

     # apt update && apt upgrade

.. note:: The system upgrade is optional, you can skip it.
